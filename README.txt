A series of helper functions to extract data from my HubSpot database.

get_engaged_contacts: Returns a dataframe of all contacts who were created or engaged after a certain date.

get_deals: Return a dataframe of all deals after a given date.

get_associated_contacts: Return a dataframe of all contacts associated with given deals.

get_customers: Return a dataframe of all paying customers.

get_email_events: Return a dataframe of all email events for a given list of email addresses.
